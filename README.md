# Puppetlabs Modules Roundup Writer
This repository helps automate some of the writing for the Puppetlabs module roundups posts.

Each post highlights the module releases of the last month, providing an overview of the changes that happened in each module.

## Template

Use the monthly Markdown template at [MONTHLY_ROUNDUP_TEMPLATE.md](MONTHLY_ROUNDUP_TEMPLATE.md) as the base input when generating a new month of updates.

## Agent Instructions (LLM)

See [AGENTS.md](AGENTS.md) for agent-specific instructions on generating monthly roundup posts.
